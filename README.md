# caldey-music
使用Vue3+TS+Pinia+TailWindCSS开发一个简约风格的响应式音乐播放网站，通过TailwindCSS实现了PC端/移动端响应式布局，全局适配骨架屏
已完成50%功能，登录功能，歌曲搜索，歌词，MV等功能暂未开放
